# Breast Cancer Classification using MaxMammoNet

## Project Description

This project presents a deep learning approach for breast cancer classification using mammogram images. The objective is to develop a reliable image classification model that can assist in identifying breast cancer by learning meaningful patterns from mammographic scans.

The project is implemented using the PyTorch deep learning framework and utilizes the MIAS (Mammographic Image Analysis Society) dataset. Various image preprocessing and data augmentation techniques are applied before training the model to improve its performance and generalization.

The notebook demonstrates the complete workflow, including data preparation, model development, training, evaluation, and visualization of the results.

---

## Objectives

The primary objectives of this project are:

- To preprocess mammogram images for deep learning.
- To apply image augmentation techniques for improving model performance.
- To build a breast cancer classification model using the MaxMammoNet architecture.
- To evaluate the trained model using standard classification metrics.
- To visualize the learning performance through training and validation curves.

---

## Dataset

This project uses the MIAS (Mammographic Image Analysis Society) dataset for breast cancer classification.

The dataset consists of mammogram images collected for research purposes and includes different categories of breast abnormalities. The images are preprocessed before being used for model training and evaluation.

The dataset is not included in this repository because of licensing and distribution restrictions.
